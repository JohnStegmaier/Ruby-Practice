|-----------------|-----------|
| New York        | 212       |

| San Francisco   | 301       |

| Palo Alto       | 650       |
| Evanston        | 847       |
| Orlando         | 407       |
| Lancaster       | 717       |

The application will list out all the city names in the dictionary and will allow you to either find an area code by city.
